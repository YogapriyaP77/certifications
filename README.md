# Breast Cancer Detection (Streamlit)

## Overview

This project predicts whether a breast image is **Cancer or Non-Cancer** using a deep learning model (ResNet18).

## How to Run

1. Install dependencies
   pip install -r requirements.txt

2. Train model
   python train.py

3. Run app
   streamlit run app.py

## Output

* Prediction: Cancer / Non-Cancer
* Confidence score

## Note

Model file is not included due to GitHub size limits. Run training to generate it.
